Overall Formatting

The overall site colors are light gray, white, and dark blue. The general font-family for the site is Arial, Helvetica, sans-serif. The general text size is 16px.

Site Header / Branding

The header uses a dark blue background with white text, centered alignment, and padding to clearly display the site title.

Navigation Links

The navigation bar uses a dark blue background with centered links. The links are white, bold, and become underlined when hovered over.

Content Section

The main content section uses a white background, centered layout, padding for spacing, and margins between sections. Section titles are styled in dark blue.

Site Footer

The footer uses a dark blue background with white centered text and includes a link to the portfolio style guide.

Layout and Positioning

The website uses a separate layout.css file for main layout styling. The header is centered with padding and a dark background. The navigation uses flexbox to organize links horizontally on larger screens and vertically on smaller screens. The main content area has a max-width, centered margin, and padding for better spacing. The footer stays at the bottom of the page using flexbox on the body.

Responsive Design

The site uses responsive web design through flexible widths, media queries, and a navigation menu that changes layout on smaller screens.
